# Momsy

AI-powered baby care assistant built with SwiftUI and Firebase.

## Features

- Baby tracking
- AI assistant
- Sleep analytics
- Feeding logs
- Growth tracking
- Multi-language support

## Tech Stack

- SwiftUI
- Firebase
- Firestore
- MVVM
- Clean Architecture

## Setup

1. Clone repository
2. Add GoogleService-Info.plist
3. Run project in Xcode

## Architecture

Project follows Clean Architecture principles.
